# Курсы

Набор курсов - Словарь таксономии, каждый термин, это отдельный курс

Нода - "Глава" - отдельная страница курса, которая заканчивается вопросами и имеет служебные характеристики, время чтения, вес, ...

Изложение материала:
 - текст
 - аудио
 - видео
 - анимированный текст

Роли:
 - ученик
 - преподаватель
 - модератор
